SMRT is a verified SMR storage stack.

## Setup

Ensure you have Dafny installed.  If the Dafny binary is not located at
`~/code/dafny/Binaries/`, set `DAFNY_ROOT` on your shell appropriately.

## Building

```
$ make -C src/
```
